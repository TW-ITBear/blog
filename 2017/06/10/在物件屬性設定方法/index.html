在物件屬性設定方法

發表於 2017-06-10 |

javascript 很隨興方便，除了可以針對物件(Object) 設定屬性(Property)以外，還可以設定方法(Method)，呼叫指定的函式(function)。

MDN 上就解釋了 Object 的各個部分，而針對設定方法也有做說明，很可惜沒有針對有參數的方法做說明，或許覺得簡單，相同處也很多所以省略 (幾乎一模一樣)，而本人遇到的需求恰巧就是要有參數的，所以記下了試驗結果。

節省時間可以看完整的程式碼

以下就針對範例各個部分做簡單說明。

  1. HTML
  2. Javascript 設定無參數 function
  3. Javascript 設定有參數 function
  4. 心得小結

HTML

範例HTML，主要是呈現結果的部分。一個放沒有參數的方法呼叫結果，另一個放有參數的方法呼叫結果。

無參數的 function

先定義好一個 function 叫做 print，之後宣告一個 Object ，並指定他的 doPrint 屬性是呼叫 print。 執行該方法的部分就是把屬性名稱當作 function 的執行方式去執行。

有參數的 function

先定義好一個 function 叫做 print，之後宣告一個 Object ，並指定他的 doPrint 屬性是呼叫 printWithParameters。 執行該方法的部分就是把屬性名稱當作 function 的執行方式去執行。

心得小結

說實在跟無參數的設定方式一樣，但是沒使用過會不知道怎麼去試。

覺得容易搞混的是，到底什麼時候加上()??

目前理解是，在綁定屬性的時候，是告訴這個屬性待會去呼叫哪個function，所以只需要 function 的名稱就好。後續就是一般在執行 function 的用法。
